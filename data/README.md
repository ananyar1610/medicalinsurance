## Medical Insurance Charges Prediction
### Project Overview

This repository contains a machine learning application that predicts medical insurance charges based on demographic and health-related attributes. The project includes data preprocessing, exploratory data analysis, multiple regression models, and a Streamlit web application for interactive predictions.
